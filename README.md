Sistema simples de dicionário para um projeto final de Programação Orientada a Objetos - II.
Consiste em um website HTML local com integração com Gemini AI que gera definições para palavras, mesmo sendo fictícias, 
e armazena-as em um banco de dados, como um dicionário online estilo UrbanDictionary.

# NOTA: Certifique-se que seu application.properties contenha A SUA CHAVE GEMINI API, e o SEU LOGIN/SENHA DE BANCO DE DADOS.
